
# To-Do List Manager (Python)

A simple command-line To-Do manager using Python Lists.  
You can add tasks, view tasks, and remove tasks.

---

## Features
- Add new tasks
- View all tasks
- Remove a task
- Menu-driven program

---

## How It Works
We use a Python **list** to store tasks because lists are mutable (we can change them anytime).
